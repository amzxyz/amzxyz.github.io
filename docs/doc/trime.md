# 🤖 Trime (同文输入法) 部署指南

欢迎在 Android 平台使用万象。同文输入法 (Trime) 是安卓端高度可定制的 Rime 前端。虽然它的文件替换逻辑与电脑端非常相似，但由于其自身特性，请务必留意以下几个核心细节。

[:octicons-download-24: 前往 GitHub Releases 获取最新版](https://github.com/osfans/trime/releases)
---

### 1. 初始化并授权目录 (Android 11+ 必看)

由于较新安卓系统的存储限制，同文不能再随意读取手机根目录：

1. 安装同文输入法后，打开 APP 主界面。
2. 根据屏幕提示，授权同文输入法**访问存储空间**的权限。
3. 按照向导，选择或新建一个用于存放配置文件的目录（通常默认是手机存储下的 `rime` 文件夹）。

---

### 2. 置入万象方案与模型

**📦 下载万象拼音方案**

* ⚡ **国内高速节点 (CNB，免翻墙)**：[:octicons-link-external-24: 点击前往下载](https://cnb.cool/amzxyz/rime-wanxiang/-/releases)

* 🌍 **国际开源节点 (GitHub)**：[:octicons-link-external-24: 点击前往下载](https://github.com/amzxyz/rime_wanxiang/releases)

**📦 下载语法模型**

* [:octicons-download-24: wanxiang-lts-zh-hans.gram](https://cnb.cool/amzxyz/rime-wanxiang/-/releases/download/model/wanxiang-lts-zh-hans.gram)

使用手机自带的文件管理器，进入 `rime` 文件夹：

1. 将下载的 **`wanxiang-lts-zh-hans.gram`** 语法模型文件放入该目录。
2. 将万象方案压缩包解压后的**所有文件和文件夹**复制进去（切勿包含外层根目录）。如果提示同名文件，请选择**全部覆盖**。

!!! info "专属键盘皮肤 (`.trime.yaml`)"
    仓库内附带了专门为同文设计的皮肤文件（如 `简纯+.trime.yaml`）。同文与电脑端不同，它的键盘按键布局、符号面板等都是由皮肤文件定义的，强烈建议您保留并使用它以获得完整的万象体验。没有提供在压缩包是因为人均自定义，接管的太多不好，有需求自行下载即可，位于仓库custom文件夹：简纯.zip。

---

### 3. 🚨 核心预警：执行重新部署与“假死”现象

回到同文输入法 APP 首页，点击 **【部署】**。

!!! danger "⚠️ 极度重要：请容忍“假死”与卡顿"
    由于同文输入法的编译机制，在首次部署时，会消耗极大的手机算力。

    * 部署时间可能长达 **2 分钟**！

    * 期间弹出键盘无响应，请不要过多测试操作，等待。屏幕弹出“部署成功”的提示文字。

---

### 4. 启用专属皮肤

部署成功后，万象的词库已经生效，接下来我们需要激活配套的键盘布局：

1. 在同文 APP 主页点击 **【主题】**。
2. 在列表中找到刚刚复制进去的万象配套皮肤（如 `简纯+`），点击选中。

---

### 5. 初始指令与输入类型切换

部署成功后，万象的默认状态如下：

* 🟢 **Base 标准版**：默认开启 **全拼**。
* 🔵 **Pro 增强版**：默认开启 **自然码双拼**。

!!! tip "强烈建议执行一次激活指令"
    即使默认方案恰好是您需要的，我们也建议您利用万象强大的斜杠指令进行一次主动切换。这一步操作能确保底层多个配置文件的协同绑定完美契合您的输入习惯。
