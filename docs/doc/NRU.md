# 🔠 字母引导模式：N日期模式 / R人名币大写 / Unicode码模式

> **大写字母，通过shift+rnu来输入，不是遇到问的多了这话我真不想多说！**

---

## 💴 R 模式：数字与财务金额大写
财务报账、填写合同的绝对效率利器。免去任何繁琐的脑内转换和错字风险。

* **触发方式**：大写 `R` 开头 + 一串阿拉伯数字。

* **实战演示**：输入 **`R1234`**，引擎会瞬间为你生成：

  1. `一千二百三十四` (标准中文数字)

  2. `壹仟贰佰叁拾肆元整` (标准财务大写)

  3. `一二三四` (纯汉字直译) 等多种格式。

!!! success "独家操作优化：极速盲选上屏"
    为了打破常规数字选词的冲突，在 R 模式下，选词体验被进行了重构：

    * **字母映射**：你可以直接敲击键盘上的 **`q, w, e, r, t, y...`** 键，它们依次代表第 1 到 6 个候选项，实现按键不位移的极速上屏。

    * **传统兼容**：当然，你依然可以使用 `↑` `↓` 方向键来上下移动焦点进行回车确认。

## 🔣 U 模式：Unicode 底层精准定字

当你知道某个生僻字或特殊符号的 Unicode 编码，但完全不知道它怎么读时，这是最直接的物理降维打击。

* **触发方式**：大写 `U` 开头 + 16 进制 Unicode 编码。

* **实战演示**：输入 **`U62fc`**，候选框会直接丢出对应的汉字「拼」。

---

## ⚡ N 模式：极速日期录入

在大写字母 `N` 后紧跟数字，即可触发安全占位逻辑的数字日期模式，彻底终结手动敲击年月日符号的烦恼。输入完毕后，可直接使用数字键上屏，同时**完美兼容 `q, w, e...` 代替数字键极速上屏**。

* **4位数模式 (月日)**：输入 `N0101` 或 `N0312`，系统会自动补全当前的年份。

* **8位数模式 (年月日)**：输入 `N20250315`，直接输出完整指定的年月日格式。

> **💡 极客定制**：N 模式同样继承了强大的自定义能力（参考时间日期），它的输出类型、格式、顺序、候选数，每一项均可在 YAML 配置文件中由你一手掌控。
