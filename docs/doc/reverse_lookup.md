# 🧩 生僻字反查 (Reverse Lookup)：无拼音定字的终极融合

> **遇到生僻字，不知道读音怎么办？不需要切出浏览器去查，万象将两分、多分与笔画重组为一个“全聚合引擎”，让你见字拆字，所见即所得。**

传统输入法的反查模式往往是割裂的：要么只能用纯笔画，要么只能用纯部首。万象的生僻字反查彻底打破了这层壁垒，构建了一个极度高集成的混合引擎。无论你怎么拆解，只要输入对应的编码组合，生僻字就会立刻现身。

---

## ⚡ 核心机制：All-in-One 融合引擎

在万象中，反查不需要专门的模式切换，它就潜伏在你的日常输入流中。

**触发方式**：在正常拼音状态下，按下引导符 **``` ` ```** (Tab 键上方)，输入框会瞬间提示 `〔反查：部件|笔画〕`。
一旦你继续输入后续字母，这个提示词会自动隐去，底层引擎立刻接管你的按键，开始“降维解析”。

### 1. 见字拆字：部件拼音的自由组合

你可以将不认识的字，拆分成你认识的两个或多个部分，然后**直接输入这些部分的拼音**。

!!! success "实战演示：遇到不会读的「震」字"
    1. 你观察到“震”是由**“雨”**和**“辰”**上下组成的。

    2. 按下反查引导符 **``` ` ```**。

    3. 继续输入“雨”和“辰”的拼音（以自然码双拼为例，输入 `yu if`）。

    4. 目标字“震”瞬间跃然屏上！

### 2. 笔画兜底：横竖撇捺折 (hspnz)

如果某个部件字形太散没法拆，你可以随时在反查编码中混入**五笔画**。

* 键盘映射：**`h`** (横)、**`s`** (竖)、**`p`** (撇)、**`n`** (捺)、**`z`** (折)。

* 你完全可以做到：打一半部件拼音，剩下一半用 `hspnz` 笔画来凑！高度自由，极度包容。

* 在万象的wanxiang_reverse.custom.yaml自定义文件中可以选择笔画的代表编码，因此`hspnz`非固定。

## 🔬 透视：反查候选词的信息金矿

反查不仅是为了帮你打出这个字，更是为了**“教你认识这个字”**。

当反查结果出现时，万象会在候选词的注释区提供例如，当你查出“震”字时，候选项后方会紧跟一段超级注释：

> **震 〔音zhèn・辅yh・基本〕**

!!! info "超级注释信息拆解"
    1. **读音 (`音zhèn`)**：直接告诉你它的正确全拼与声调。下次遇到它，你就可以直接用拼音盲打了。

    2. **辅助码 (`辅yh`)**：提取并展示该字的专属辅助码（例如 `y` 代表雨的声母，`h` 代表辰的声母）。学会了辅码，以后重码时就能一击必中。

    3. **编码区 (`基本`)**：告诉你这个字属于 Unicode 的哪个汉字区块（基本区、扩展A-I区等）。极客玩家的最爱。

## 🛠️ 底层依赖与方案匹配 (Custom 调优)

为了让反查的输入体验与你日常的打字节奏“无缝咬合”，万象的反查引擎是**高度可定制**的。

!!! note "配置解析：`wanxiang_reverse` 方案"
    反查功能的底层依赖于根目录下的 `wanxiang_reverse.schema.yaml` 方案文件。
    
    默认情况下，如果你主方案用的是自然码双拼，那么反查时输入部件也应该遵循自然码的规则（比如“金”打 `jn`）。但如果你想修改反查时的键位映射，可以通过创建或编辑 `wanxiang_reverse.custom.yaml`，将其配置成任意的全拼或特定的双拼方案。
    
    **一句话总结：主方案怎么打，反查方案就怎么配，保证一致，指法绝不割裂。**
