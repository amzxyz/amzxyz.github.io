# 核心输入体系：万象拼音生态

万象拼音（Wanxiang）并非仅仅是一个简单的 Rime 配置文件，而是一套为 Rime 引擎量身打造的 **全场景拼音解决方案**。

它从最底层的“带声调词库”与“八股文语法模型”开始重构，涵盖了输入方案、智能预测、多音字精细化处理等全流程，旨在为您提供由内而外的极致拼音输入体验。

---

## 🧠 引擎心脏：RIME-LMDG 语法模型

万象之所以能实现“智能整句输入”，其核心引擎来自于万象生态的开源姊妹项目：**[RIME-LMDG](https://github.com/amzxyz/RIME-LMDG)**。它为 Rime 注入了强大的语言理解能力。

* **📚 海量语料**：基于 32GB 多领域语料库（涵盖新闻、法律、文学、社交日常等）深度训练。
* **⚙️ 算法架构**：采用多级 n-gram 语言模型，并针对“词 + 拼音”双键进行统计计算。
* **🎯 极致纠偏**：实现了真正的**精准多音字处理**（轻声、变调发音均与国家幼儿园最新课本对齐）。
* **🔠 全字集覆盖**：从 CJK 基本汉字区一路支持至生僻字扩展 G 区。

!!! tip "💡 模型命中率与兼容性说明"
    LMDG 模型理论上兼容任何编码形式的 Rime 方案。但由于其核心基于**拼音和声调**进行优化，因此智能预测的命中率表现为：
    `带声调拼音方案 (如万象) > 传统无调拼音 > 形码整句 > 词组类方案`。

## ⚖️ 矩阵选择：标准版 vs 增强版

万象主仓库 **[rime-wanxiang](https://github.com/amzxyz/rime-wanxiang)** 提供了两个截然不同的方案分支。请根据您的打字习惯，选择最适合您的基石：

| 对比维度 | 🟢 万象标准版 (Base) | 🔵 万象增强版 (Pro) |
| :--- | :--- | :--- |
| **核心定位** | 极致顺滑的传统大厂平替 | 硬核玩家的极限调优利器 |
| **适合人群** | **全拼/双拼用户**，首次接触 Rime 或刚从搜狗/微信输入法迁移的用户。 | **双拼进阶用户**，追求极致盲打与首选率，熟练掌握一种或多种辅助码的用户。 |
| **核心编码** | 声调全拼 | 声调全拼 + 独立辅助码 |
| **双拼支持** | 支持任意双拼方案挂接。 | 内置 10 种主流双拼（自然码/小鹤/微软/搜狗/紫光/智能ABC/国标/自然龙等），可任意切换。 |
| **辅码支持** | ❌ 不支持 | ✅ 内置 7 种辅码体系（自然码/小鹤/墨奇/汉心/五笔/仓颉/虎码首末），搭配组合千变万化。 |
| **特色反查** | 支持单字状态下 ` 引导反查；支持部件/笔画组词反查。 | 同标准版。 |
| **生僻字注音**| 开启后，输入区与候选项实时展示带声调拼音。 | 同标准版。 |

*(👉 **新手建议：** 如果您刚刚开启 Rime 的探索之旅，请毫不犹豫地从 **标准版 (Base)** 开始！)*

## 🛡️ 理念边界与开发者寄语

万象致力于夯实 Rime 生态中的“基础底座”。但为了避免您的预期偏差，我们必须坦诚地说明**本项目“不做什么”**：

!!! warning "关于学习成本的郑重声明"
    1. **这里没有输入法基础教学**：本项目不负责教您“如何记忆小鹤双拼键位”或“什么是五笔字根”。掌握双拼和辅助码是您提升效率的必经之路，这需要您在兴趣的驱动下自行调研与练习。
    2. **不提供保姆级 Rime 扫盲**：全网已有海量关于 Rime 引擎基础配置的优秀教程。使用万象需要您具备最基础的折腾精神和软件认知能力。

我们将把所有有限的精力，持续聚焦于**优化模型算法、提纯优质词库**，力求在底座层面为您提供最坚实的后盾。
