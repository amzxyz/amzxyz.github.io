    万象输入快捷键

    一、输入模式切换（全局）

    快捷键 功能
    Control+Shift+3 / Control+Shift+# 切换中英文标点
    Control+Shift+4 / Control+Shift+$ 切换简繁转换
    Control+e 切换翻译模式（中英互译）
    Control+a 切换辅助码/声调提示显示
    Control+s 切换输入码显示为带声调拼音
    Control+t 切换超级提示
    Control+g 切换字符集过滤（大字集/小字集）
    Shift+Space 切换到下一个输入方案

    二、候选菜单操作

    快捷键 功能
    -（减号） 向上翻页
    =（等号） 向下翻页
    Tab 在拼音音节间向右跳转
    Control+Tab 上屏当前候选并自动进入补码模式
    Control+j 上移当前高亮候选（手动排序）
    Control+k 下移当前高亮候选
    Control+l 重置候选位移（取消排序调整）
    Control+p 置顶当前候选
    Control+Delete 删除或降权当前候选词

    三、编码区编辑

    快捷键 功能
    Alt+Left 光标左移一个音节
    Alt+Right 光标右移一个音节
    Control+w 删除光标前一个词
    Tab 光标右移一个音节
    Control+Tab 上屏首字并依次补码

    四、通用编辑器操作

    快捷键 功能
    Space 上屏当前候选项
    Return 上屏原始输入
    Control+Return 上屏经转换后的内容（如带声调拼音）
    Control+Shift+Return 上屏注释内容
    BackSpace 删除上一个字符
    Delete 删除后一个字符
    Control+BackSpace 删除一个完整音节
    Escape 取消当前输入

    五、光标导航（navigator）

    快捷键 功能
    Left 左移一个字符
    Right 右移一个字符
    Shift+Left 左移一个音节
    Shift+Right 右移一个音节

    六、特殊输入引导（输入编码前缀）

    前缀 功能
    `（反引号） 部件拆字/笔画反查
    U + 十六进制 Unicode 字符输入（如 U4E00 → 一）
    R + 数字 数字金额大写转换（如 R123 → 壹佰贰拾叁）
    V + 表达式 计算器功能（如 V1+2 → 3）
    / 或 o + 命令 日期/时间/农历等（见下表）

    日期时间命令（前缀 / 或 o）

    命令 功能
    /sj 或 osj 时间
    /rq 或 orq 日期
    /nl 或 onl 农历
    /xq 或 oxq 星期
    /ww 或 oww 今年第几周
    /jq 或 ojq 节气
    /dt 或 odt 日期+时间
    /tt 或 ott 时间戳
    /jr 或 ojr 节日
    /day 或 oday 问候模板
    N20250315 大写 N + 日期 → 格式化日期
    /htj 输入统计（时光机）

    输入统计命令

    命令 功能
    /tj 总统计
    /rtj 今日统计
    /ztj 本周统计
    /ytj 本月统计
    /ntj 本年统计
    /qctj 清除数据
    /htj20231015 历史统计（年月日递进）
    /htj20231015t20240102 区间统计

    n/ 《 m/ 》

    八、成对符号包裹

    输入 内容 + \ + 字母 → 用符号包裹候选词（如 nihao\c → [你好]）

    常用包裹符号表

    字母 符号 字母 符号 字母 符号
    q () w （） a []
    s ［］ z {} x ｛｝
    g 「」 i 『』 k 《》
    l 〈〉 j <> d 〔〕

    Markdown 快捷包裹

    输入 功能
    \md 或 \jc 粗体 **文字**
    \it 斜体 __文字__
    \st 删除线 ~~文字~~
    \ln 行内代码 `文字`
    \cb 代码块 ```文字```
    \lk 链接 [文字](url)
    \im 图片 ![文字](img)
    \h ~ \hhhh 一级到四级标题 # 文字

    九、英文输入相关

    功能 说明
    \（反斜杠，双击） 英文造词符号
    英文自动加空格 配置了 smart 智能模式，连续输入英文时自动添加空格

    十、快捷键冲突说明

    • 部分快捷键（如 Control+Shift+3）可能被系统或其他软件占用，可自行修改配置文件中的 accept 值。
    • 快捷键是否生效取决于当前状态：是否在输入编码、是否有候选菜单等。
