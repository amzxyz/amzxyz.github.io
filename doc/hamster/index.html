    📱 Hamster (仓输入法) 部署指南

    欢迎在 iOS 平台使用万象。仓输入法是目前苹果生态中最优秀的 Rime 前端之一。

    ⚠️ iOS 平台特殊机制预警

    请注意，iOS 系统存在严格的沙盒隔离机制。仓输入法将文件分为 “键盘文件”、“应用文件” 和 “iCloud 文件”。区分这些文件层级是 iOS 平台的特性，与万象方案本身无关。请务必仔细阅读本教程中的文件同步与覆盖步骤，切勿跳过。

    1. 启用键盘与完全访问权限

    1. 安装好仓输入法 APP 后，进入 iOS 系统的 设置 → 通用 → 键盘 → 键盘 → 添加新键盘，选择并添加 Hamster。
    2. 点击刚才添加的 Hamster，必须开启“允许完全访问”。 (如果不开启此项，万象 Pro 版本的双拼类型切换、Lua 脚本运行等高级功能将无法正常工作。)

    2. 获取并置入方案文件

    您可以选择 APP 内直接下载方案，但语法模型（gram 文件）必须手动下载置入。

    方案一：APP 内快捷下载 (推荐)

    1. 打开仓输入法 APP 主页，进入 【输入方案设置】 页面。
    2. 点击右上角的 + 号，搜索 “万象”，根据您的需求（Base 或 Pro）直接下载对应的方案文件。
    3. 手动下载语法模型：前往下载 wanxiang-lts-zh-hans.gram，打开 iOS 自带的 文件.app，找到路径 我的iPhone/Hamster/Rime，将该 gram 文件放入此目录。

    方案二：局域网/手动全量导入

    如果您觉得在手机上解压和移动文件太麻烦，推荐使用 Wi-Fi 传书： 1. 确保手机和电脑连接到同一个 Wi-Fi。 2. 在仓输入法 APP 内点击 【Wi-Fi 上传方案】，获取【局域网访问地址】。 3. 在电脑浏览器中打开该地址，将电脑上下载并解压好的万象方案所有文件（不包含外层根目录）以及 gram 语法模型，全部拖拽上传进去。

    3. 🚨 核心避坑：iCloud 同步与部署顺序

    在点击部署之前，请先确认您是否在仓输入法中开启了 iCloud 同步功能，这直接决定了您的部署是否会成功。

    如果您开启了 iCloud 同步 (极易踩坑！)

    切记：上述所有的下载文件都应该放置在 iCloud 对应的 Hamster 文件夹中！

    如果您是通过 APP 内搜索下载 的方案，下载完成后千万不要直接点部署！ 1. APP 内下载的文件默认存放在“应用文件”中（具体文件管理可参考仓输入法官方文档）。 2. 请先去 APP 的文件管理中，将这些文件复制到 iCloud 目录中。 3. 原因：如果您没复制就直接点部署，iCloud 中的旧文件会直接反向覆盖掉您刚下载的新文件，相当于您完全没有成功下载万象方案！

    如果您未开启 iCloud 同步

    不管您是手动下载还是 APP 内下载，方案都会老老实实呆在“应用文件”中，您可以跳过同步步骤，直接进行下一步的部署。

    4. 执行初次部署

    在仓输入法 APP 主页面往下翻，找到并点击 【RIME】 → 点击 【重新部署】。

    ⏳ 等待真空期：首次部署需要 1~2 分钟。注意：第一次部署成功后，可能依然需要等待一段时间才能开始使用，这段时间内如果在键盘上打字，将不会有任何候选项，此为正常现象，请耐心等待。

    5. 皮肤与斜杠按键设置 (processByRIME)

    为了让万象的命令系统正常工作，皮肤的设置至关重要。

    1. 推荐使用 Release 发布页中提供的万象专用皮肤，并务必阅读皮肤的使用说明。
    2. 核心设置：在仓输入法中，输入斜杠 / 时绝对不能直接上屏，必须将其设置为交由 Rime 处理 (processByRIME)。只有这样，在中文输入状态下不上屏地输入完命令，万象才能识别它。
    3. 如果您使用的是第三方自定义键盘皮肤，请务必自行在 APP 的皮肤 UI 编辑界面中，为斜杠按键添加“由 Rime 处理”的属性。

    6. 输入类型切换与“反向同步”机制

    万象默认状态为：Base 版默认全拼，Pro 版默认自然码。即使默认方案符合您的习惯，我们也强烈建议您执行一次切换指令。

    第一步：在键盘上输入指令

    唤出仓输入法键盘，直接打字输入以下对应指令：

    /flypy    → 小鹤双拼
/mspy     → 微软双拼
/zrm      → 自然码
/sogou    → 搜狗双拼
/znabc    → 智能ABC
/ziguang  → 紫光双拼
/pyjj     → 拼音加加
/gbpy     → 国标双拼
/lxsq     → 乱序17
/pinyin   → 全拼
/zrlong   → 自然龙(反查是全拼)
/hxlong   → 汉心龙(反查是全拼)
/jjf      → 间接辅助
/zjf      → 直接辅助

    第二步：🚨 必须执行的“反向同步”操作 (绝不可漏)

    当您在键盘上输入完 /zrm 并生效后，这个配置仅仅保存在了“键盘文件”中！ 为了保证下次部署不丢失该配置，您必须手动将其同步回 APP：

    1. 打开 仓 APP → 文件管理 → 通用。
    2. 点击 【使用键盘文件覆盖应用文件】。
    3. (仅限开启 iCloud 的用户)：还要再去 仓 APP → iCloud 同步 中，点击 【拷贝应用文件至 iCloud】。
    4. 完成上述备份后，回到 APP 首页，再次点击 【重新部署】。

    只有完成这套“反向同步”流程，您的切换操作才算真正永久生效！
