    ⭕ HarmonyOS NEXT (超越输入法) 部署指南

    欢迎在纯血鸿蒙 (HarmonyOS NEXT) 平台使用万象。由于系统架构的革新，传统的安卓输入法已无法使用，超越输入法 是目前在该平台上运行 Rime 引擎的首选利器。

    请仔细阅读以下专为鸿蒙平台优化的“ZIP 整包导入”部署流程。

    1. 下载并安装前端应用和方案文件

    请直接在华为手机的 应用市场 (AppGallery) 中搜索并下载 【超越输入法】。

    在开始安装前，请确保您已经下载了最新的前端软件和万象方案包：

    📦 下载万象拼音方案

    请根据您的网络环境选择下载源，前往 Releases 页面获取最新版方案压缩包（.zip）：

    • ⚡ 国内高速节点 (CNB，免翻墙)： 点击前往下载

    • 🌍 国际开源节点 (GitHub)： 点击前往下载

    📦 下载语法模型

    • wanxiang-lts-zh-hans.gram

    2. 在电脑端准备 ZIP 部署包

    与 Windows/Mac 直接拖拽文件不同，超越输入法需要通过导入 ZIP 压缩包的方式来读取方案和皮肤。

    1. 准备方案包 (rime-wanxiang-base.zip)：
    2. 新建一个文件夹，命名为 Rime。
    3. 将万象方案解压出来的所有文件、文件夹，以及 wanxiang-lts-zh-hans.gram 语法模型，全部放进这个 Rime 文件夹中。

    4. 然后将这个 Rime 文件夹打包压缩成 Rime.zip。

    5. 准备皮肤包 (VGHFly.zip)： 请直接点击下方按钮获取万象专属的鸿蒙主题皮肤包：

    点击下载 VGHFly「万象」皮肤包

    3. 传输文件至手机

    将打包好的方案包 Rime.zip 和皮肤包 VGHFly.zip，通过微信、互传或数据线，全部移动到手机本地存储的 Download (下载) 目录中。

    4. 在应用内导入与激活

    打开手机上的【超越输入法】APP，按照以下步骤进行操作：

    1. 导入方案：找到方案导入入口，选中 Download 目录下的 Rime.zip 作为方案导入。
    2. 导入皮肤：找到皮肤导入入口，选中 VGHFly.zip 作为皮肤导入。
    3. 选择并启用：在应用的方案列表和主题列表中，分别勾选您刚才导入的万象方案和皮肤。

    5. 执行重新部署

    选择完毕后，查看方案列表中是否已经出现了“万象拼音（或 万象拼音pro）”的选项。

    🔄 如果没有出现或需要使其生效：请点击页面右上角的 【刷新】 按钮，超越输入法将开始执行 Rime 的部署编译流程。请耐心等待部署完成。

    关于皮肤深度定制

    本教程提供的是开箱即用的主题包。如果您感兴趣，想要针对鸿蒙进行高度个性化的皮肤定制，请务必直接查阅 [超越输入法官方文档]。超越的皮肤参数与电脑端、同文端均不通用。但与同文语法相似。

    6. 初始指令与个性化切换

    部署成功后，万象的默认状态如下：

    • 🟢 Base 标准版：默认开启 全拼。
    • 🔵 Pro 增强版：默认开启 自然码双拼。

    强烈建议执行一次激活指令

    即使默认方案恰好是您需要的，我们也建议您利用万象强大的斜杠指令进行一次主动切换。这能确保底层多个配置文件的协同绑定完美契合您的输入习惯。

    👉 操作方法：在任意输入框（中文状态下），直接打字输入以下对应指令，然后回到 APP 再次点击【刷新】（重新部署） 即可生效。

    /flypy    → 小鹤双拼
/mspy     → 微软双拼
/zrm      → 自然码
/sogou    → 搜狗双拼
/znabc    → 智能ABC
/ziguang  → 紫光双拼
/pyjj     → 拼音加加
/gbpy     → 国标双拼
/lxsq     → 乱序17
/pinyin   → 全拼
/zrlong   → 自然龙(反查是全拼)
/hxlong   → 汉心龙(反查是全拼)
/jjf      → 间接辅助
/zjf      → 直接辅助
