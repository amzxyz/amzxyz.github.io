    🤖 Fcitx5-Android (小企鹅) 部署指南

    欢迎在 Android 平台使用万象。Fcitx5 (小企鹅输入法) 在安卓端同样支持 Rime 引擎。由于高版本安卓系统的权限机制较为严格，请务必仔细阅读以下避坑指南。 前往 GitHub Releases 获取最新版

    注: 此为魔改版本，也是可定义键盘的体验最好的版本，因此极为推荐。

    1. 安装与初始化文件夹

    1. 安装完 Fcitx5 应用后，请先在应用内添加 【中州韵】 (Rime) 插件。
    2. 添加插件后，小企鹅会自动在系统目录中生成一个对应的 Rime 用户配置文件夹。

    ⚠️ 严禁手动创建该文件夹

    如果误删除了应用自动生成的 Rime 文件夹，自行手动建立一个同名文件夹是无效的！系统将无法识别。您必须在 Fcitx5 设置中重新配置或重新添加插件，让应用自动生成合法路径。

    2. 定位 Rime 用户目录

    请使用中州韵设置-用户目录的方式跳转-安卓文件管理器（推荐使用系统自带），找到并打开以下绝对路径： 

    /storage/emulated/0/Android/data/org.fcitx.fcitx5.android/files/data/rime/
    其实前半部分绝对路径我们无需关心，进入目录后看到的是data内部，可能存在pinyin文件夹，rime则必须位于data/rime

    3. 🚨 核心避坑：置入方案与权限限制

    将万象方案解压后的所有文件（切勿包含外层压缩包根目录）以及 语法模型 wanxiang-lts-zh-hans.gram 文件，全部放入上述的 Rime 目录中。

    (MT 管理器慎用)

    由于该目录位于安卓的 Android/data 目录下，受极严格的 Linux 权限控制。

    • ❌ 错误做法：使用 MT 管理器等拥有越权能力的第三方工具直接强行复制文件。这会导致文件所有者/权限组错乱，引发读取失败，导致输入法部署报错、极其卡顿、甚至提示内存溢出 (OOM)。
    • ✅ 正确做法：
      1. 优先使用 Fcitx5 App 自带的 “导入文件” 功能（它会调用安卓标准存储框架 SAF，确保权限正确）。
      2. 或者使用手机自带的官方文件管理器进行常规复制。
      3. (Root 用户专属)：如果非要强行复制，完成后务必手动修正所有相关文件的 chown 和 chmod 权限，使其与 Fcitx5 应用的权限组保持一致。

    关于皮肤文件的说明

    压缩包内附带的如 简纯+.trime.yaml 等皮肤文件，是专门给“同文输入法 (Trime)”用的。小企鹅无法读取 trime 原生的皮肤配置，复制进去也不会生效。小企鹅的外观由应用自身统一接管。

    4. 执行重新部署

    回到 Fcitx5-Android 应用主界面。第一次则可能是 【重载配置】 也能触发自动部署，后续使用一般可点击 【重新部署】

    ⏳ 耐心等待：受限于移动端处理器的性能，首次部署需要编译极大规模的词库和 Lua 脚本，可能需要等待 1~2 分钟。在出现部署成功的提示前，请不要唤出键盘操作。

    5. 初始指令与个性化切换

    部署成功后，万象的默认状态如下：

    • 🟢 Base 标准版：默认开启 全拼。
    • 🔵 Pro 增强版：默认开启 自然码双拼。

    强烈建议执行一次激活指令

    即使默认方案恰好是您需要的，我们也建议您利用万象强大的斜杠指令进行一次主动切换。这一步操作能确保底层多个配置文件的协同绑定完美契合您的输入习惯。

    👉 操作方法：直接在任意输入框打字输入以下对应指令，然后再回 APP 点击一次 【重新部署】。

    /flypy    → 小鹤双拼
/mspy     → 微软双拼
/zrm      → 自然码
/sogou    → 搜狗双拼
/znabc    → 智能ABC
/ziguang  → 紫光双拼
/pyjj     → 拼音加加
/gbpy     → 国标双拼
/lxsq     → 乱序17
/pinyin   → 全拼
/zrlong   → 自然龙(反查是全拼)
/hxlong   → 汉心龙(反查是全拼)
/jjf      → 间接辅助
/zjf      → 直接辅助
